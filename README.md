# Java Login Logout System

A simple login and logout system built using Advanced Java concepts.

## Features
- User authentication
- Session management
- Logout functionality

## Technologies Used
- Java
- Servlets
- JSP
- Eclipse IDE
- Apache Tomcat

## Purpose
This project was built to understand backend authentication and session handling.

## How to Run
1. Import project into Eclipse
2. Configure Tomcat server
3. Run on server
